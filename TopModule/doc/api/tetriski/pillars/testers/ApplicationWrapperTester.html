ApplicationWrapperTester 

class ApplicationWrapperTester extends PeekPokeTester[TopModuleWrapper]

A base class of testers for applications when the top design has a wrapper. It help users to construct the simulation processes and produce classes in the specific format of Chisel testers using the Verilator backend. We suggest users test their own CGRA architectures with this packed class.
