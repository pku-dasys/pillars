SynthesizedModule 

class SynthesizedModule extends Module

The synthesized design for a mapped DFG. When using this module, the origin CGRA architecture plays the role of virtual overlay. Some may call it "soft CGRA", which is used for quick and high-quality compiler.

This Module is only tested when II = 1.
